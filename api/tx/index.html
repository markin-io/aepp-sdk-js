Tx([options]) ⇒ Object ⏏

Basic Tx Stamp

Attempting to create instances from the Stamp without overwriting all abstract methods using composition will result in an exception.

Tx is one of the three basic building blocks of an Ae client and provides methods to create aeternity transactions.

Kind: Exported function
Returns: Object - Tx instance
rtype: Stamp

Param Type Default Description
[options] Object {} Initializer object
