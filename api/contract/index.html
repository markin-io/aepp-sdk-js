ContractBase([options]) ⇒ Object ⏏

Basic Contract Stamp

This stamp include api call's related to contract functionality. Attempting to create instances from the Stamp without overwriting all abstract methods using composition will result in an exception.

Kind: Exported function
Returns: Object - Contract instance
rtype: Stamp

Param Type Default Description
[options] Object {} Initializer object

Get contract ACI

Kind: instance abstract method of ContractBase
Returns: Object - - Contract aci object
Category: async
rtype: (source: String, options: Array) => aciObject: Promise[Object]

Param Type Default Description

[options] Object {} Options
[options.filesystem] Object Contract external namespaces map
[options.backend] Object Contract vm(default: aevm)
